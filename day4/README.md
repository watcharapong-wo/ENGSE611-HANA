## การติดตั้งและรัน

สำหรับแต่ละโปรเจกต์ ให้เข้าไปในโฟลเดอร์และรันคำสั่ง:

```bash
npm install
npm start
```

หรือสำหรับ development:

```bash
npm run dev
```

## สิ่งที่เรียนรู้

- การติดตั้งและใช้งาน Node.js
- สร้าง HTTP Server
- Express.js Framework
- Routing และ Middleware
- การจัดการ Request และ Response
- การทำงานกับ JSON
- RESTful API Design
